Para instalar React y React DOM usamos el comando:

npm install react react-dom --save

Eso instala automáticamente la última versión de estas tecnologías.

Si quieres usar otra versión, usamos el símbolo @ y la versión con la que queramos trabajar (por ejemplo, en este curso usaremos React 17, así que la instalación puede realizarse con el comando 
npm install react@17 react-dom@17 --save
